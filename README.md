# BigBasket Fullstack Clone

A full-stack e-commerce grocery application built using MERN stack.

## 🚀 Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB

## 📂 Project Structure
bigbasket-fullstack/
│
├── backend/
└── frontend/

## ▶️ How to Run Locally

### Backend
cd backend  
npm install  
node server.js  

### Frontend
cd frontend  
npm install  
npm start
